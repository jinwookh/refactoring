## 목표
- 책에 나오는 개선 사항을 실제로 회사 코드에 적용해보기
- 챕터당 최소 하나 씩 적용

## 진행 방식
- 강의를 기반으로 스터디 진행
- 책은 본인 선택
- 총 7주 계획
- 한 주 대략 4챕터씩
